# Tinder Clone em Flutter

Um projeto de estudo desenvolvido em **Flutter** com o objetivo de reproduzir a experiência visual e parte da navegação de um aplicativo no estilo **Tinder**.

Este repositório foi criado para praticar construção de interfaces modernas, animações, composição de widgets e organização de telas em Flutter.

---

## 📱 Sobre o projeto

O **Flutter Tinder Clone** é uma aplicação focada em **UI/UX**, inspirada no layout de apps de relacionamento, trazendo uma experiência moderna, fluida e visualmente atraente.

A proposta do projeto é simular elementos comuns desse tipo de aplicativo, como:

- cartões de perfil
- navegação intuitiva
- botões de ação
- animações de interação
- interface moderna e responsiva

Este projeto é ideal para quem deseja estudar **Flutter UI**, **estruturação de telas**, **componentes reutilizáveis** e **boas práticas de organização**.

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/flutter-tinder-clone.git
```

### 2. Acesse a pasta do projeto

```bash
cd flutter-tinder-clone
```

### 3. Instale as dependências

```bash
flutter pub get
```

### 4. Execute a aplicação

```bash
flutter run
```
